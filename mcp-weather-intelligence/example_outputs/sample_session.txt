KEY INSIGHT FOR INTERVIEWERS:

This project demonstrates Model Context Protocol (MCP) — the standard for
connecting AI agents to external tools and real-time data sources.

The agent does NOT just return weather data. It:
1. Understands the user's actual goal (meetings, zoo visit, travel planning)
2. Calls the appropriate weather tools via MCP during reasoning
3. Reasons across multiple locations and timeframes simultaneously
4. Returns a specific, actionable recommendation — not a weather report

The same MCP architecture connects to any enterprise data source:
ERP systems, logistics APIs, market data feeds, inventory systems,
CRM platforms — whatever the business needs in real time.

This is the pattern every enterprise AI system uses to stay current
with live data rather than relying on static training data.
